# E50

First from-scratch LLM training run. Two phases: Phase 1 trains a small transformer on the complete works of Shakespeare (Project Gutenberg ebook #100) as a full data-pipeline dress rehearsal. Phase 2 trains the actual portfolio model on a hand-curated Lovecraft + Clark Ashton Smith corpus. Code + scripts + cleaning pipeline live here. Training runs on a dedicated Ubuntu T14; this repo is the bridge.

Pre-execution audit lives in `notes/RESEARCH_NOTES.md`.
